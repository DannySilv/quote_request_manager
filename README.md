# Quote Request Manager

A small PHP, MySQL and JavaScript project for managing quote requests.

## Goal

Small project to recall older skills in order to move to WordPress, Laravel and Vue.

It covers:

- PHP basics
- OOP PHP
- MySQL
- PDO
- Prepared statements
- CRUD operations
- HTML forms
- JavaScript validation
- Fetch API
- JSON API endpoints

## Features

Planned features:

- Quote request form
- Backend validation
- Frontend validation
- MySQL persistence
- Admin request list
- Request detail page
- Delete request
- JSON API
- JavaScript fetch integration

## Tech Stack

- PHP
- MySQL
- HTML
- CSS
- JavaScript
